消息协议格式
    version:1 byte 版本号,当前固定为1
    type:1 byte 消息类型
        0:PING请求
        1:PONG
        2:AUTH 认证消息
        3:BROADCAST_ADD 广播增加或者更新内容,订阅了此消息的用户都能看到消息更新
        4:BROADCAST_DEL 广播删除内容,订阅了此消息的用户都能看到消息更新
        4:PUSH 向指定的订阅者推送消息
        5:PULL 拉取消息
    id:2 bytes 本次请求的消息ID
    message_name_length:2 bytes 消息名长度
    message_content_length:2 bytes 内容长度
    session_id:16 bytes
    CONTENT:VARIABLE


各种类型消息格式:
TYPE AUTH REQUEST:
    message_name_length:0
    message_content_length:16
    session_id:全零
    CONTENT:
        TOKEN_ID

TYPE AUTH RESPONSE:
    message_name_length:0
    message_content_length:4
    session_id:服务器分配的会话ID
    CONTENT:
        is_error:4 bytes,0表示未发生故障,1表示认证错误,2表示协议故障

TYPE PING AND PONG:
    message_name_length:0
    message_content_length:0
    session_id:session_id

TYPE BROADCAST_ADD,PUSH,BROADCAST_DEL,PULL REQUEST
    message_name_length:VARIABLE
    message_content_length:VARIABLE
    session_id:session_id

TYPE BROADCAST_ADD,PUSH,BROADCAST_DEL,PULL RESPONSE
    message_name_length:VARIABLE
    message_content_length:VARIABLE
    session_id:session_id

    CONTENT:
        is_error:4 bytes,0表示位发生错误,1表示目标消息对象不存在,2表示系统发生错误,3表示协议错误
        message_content:VARIABLE
